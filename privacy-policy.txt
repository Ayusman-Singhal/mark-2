PRIVACY POLICY FOR Document Converter

Last Updated: September 11, 2025

OVERVIEW
--------
Document Converter ("we," "our," or "us") is committed to protecting your privacy. This Privacy Policy explains how our desktop application collects, uses, and safeguards information when you use our system diagnostic and repair tool.

INFORMATION WE COLLECT
----------------------

1. SYSTEM INFORMATION
   - Hardware specifications (CPU, RAM, storage, graphics)
   - Operating system version and build information
   - Installed software and drivers
   - System performance metrics
   - Network configuration details
   - Running processes and services

2. DIAGNOSTIC DATA
   - System logs and error reports
   - Performance benchmarks
   - Registry scan results (when using repair tools)
   - File system health information
   - Security scan results

3. USER-CREATED DATA
   - Files you choose to process (PDF tools, Excel operations)
   - System reports you generate and save
   - Custom settings and preferences
   - Repair history and audit logs

HOW WE USE YOUR INFORMATION
---------------------------

We use the collected information to:
- Provide system diagnostic and repair services
- Generate comprehensive system health reports
- Optimize system performance recommendations
- Troubleshoot hardware and software issues
- Maintain audit logs of performed actions
- Improve our diagnostic algorithms

DATA STORAGE AND SECURITY
-------------------------

LOCAL STORAGE:
- All diagnostic data is stored locally on your device
- No personal or system information is transmitted to external servers
- Data is encrypted using industry-standard encryption methods
- You maintain full control over your diagnostic reports and logs

SECURITY MEASURES:
- Application runs with appropriate system permissions only
- No unauthorized access to sensitive system areas
- Secure file handling for user documents
- Protection against malware during system scans

THIRD-PARTY SERVICES
-------------------

Our application may interact with:
- Windows System APIs for diagnostic information
- Local PDF processing libraries (no external transmission)
- Excel file manipulation tools (processed locally)
- System registry for repair operations (with user consent)

No data is shared with third-party services or external servers.

YOUR RIGHTS AND CHOICES
-----------------------

You have the right to:
- Access all diagnostic data collected about your system
- Delete any reports or logs generated by the application
- Disable specific diagnostic features
- Export your system reports in various formats
- Uninstall the application and remove all associated data

DATA RETENTION
--------------

- Diagnostic reports: Retained until manually deleted by user
- System logs: Automatically purged after 30 days (configurable)
- Audit trails: Kept for compliance purposes (user configurable)
- Temporary files: Automatically cleaned on application exit

CHILDREN'S PRIVACY
-----------------

Document Converter is not intended for use by children under 13. We do not knowingly collect personal information from children under 13 years of age.

CHANGES TO THIS POLICY
---------------------

We may update this Privacy Policy from time to time. When we do, we will:
- Update the "Last Updated" date at the top of this policy
- Notify users through the application interface
- Maintain previous versions for reference

COMPLIANCE
---------

This application complies with:
- General Data Protection Regulation (GDPR) where applicable
- California Consumer Privacy Act (CCPA) where applicable
- Microsoft Store Privacy Requirements
- Windows Application Security Standards

OFFLINE OPERATION
-----------------

Document Converter operates entirely offline. No internet connection is required for core functionality, and no personal or system data is transmitted over the internet without explicit user action (such as manually sharing a report).

DATA MINIMIZATION
-----------------

We collect only the minimum data necessary to provide diagnostic and repair services. Users can customize which diagnostic modules to run and what information to include in reports.

USER CONSENT
------------

By installing and using Document Converter, you consent to the collection and use of information as described in this Privacy Policy. You may withdraw consent at any time by uninstalling the application.
